숫자와 문자

-수의 표현 : C와 같음, 연산자도 마찬가지

-수의 연산 : Math 클래스를 이용하여 가능

(Math)
.pow( , ) : 제곱
.round( )  : 반올림
.ceil( ) : 올림
.floor( ): 내림
.sqrt( ) : 제곱근
.random( ) : 난수

-문자의 표현 : " ~ " , ' ~ '  (C와 같음) 
 typeof ~ : 자료형 파악가능

-문자연산 

" 문자 " + " 문자 " => "문자문자"

"code".length => 4
"code".indexof(c) => 1
"code".indexof(d) => 3
(생활코딩 웹사이트에 문자열 가공 예약어 모음집 있음)
