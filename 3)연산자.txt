-비교 연산자

== : 동등 연산자

ex: alert(1==2); ==> false
    alert("one"=="one") ==> true
    alert(1 == "1") ==> true    (타입이 달라도 값의 의미가 같기때문에 나온 결과)

=== : 일치 연산자 ; 좌항과 우항이 정확하게 같으냐를 판단

ex: alert(1 === "1") ==> false   (데이터 타입이 달라서 나온 결과)
